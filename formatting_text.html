Formating Text

Sometimes we need to format text in a particular way. Some words need to be bold which uses a strong tag. And, sometimes you might need to emphasise something with Italics which uses the em tag. And once, in a while you might want to underline something which uses the span tag.

The p tags does naturally make a line break but sometimes we want something more...


An extra br / tag adds a new line or a line break.

Sometimes you might want proper emphasis though on your break in which case maybe try the hr tag.

This gives a pretty cool proper line break.
